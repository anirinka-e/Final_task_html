Git

Рабочий процесс

  • git add - добавление файлов в индекс
  • git status - проверка статуса репозитория
  • git commit добавление файлов в репозиторий
  • git branch создание новой ветки
  • git checkout - переключение между ветками

Работа с файлами и папками в проекте через терминал Git

  • pwd - просмотр вашего текущего местоположения.
  • cd - переход в другую папку.
  • ls - список папок и файлов в текущей директории, где была выполнена команда.
  • mkdir - создать папку с указанным именем.
